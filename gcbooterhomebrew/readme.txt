GCBooter Homebrew v1.1 by Mega Man

Info:
This is a simple application which allows anyone to boot their GC homebrew from Wii mode.
It works from the homebrew channel.

Usage: Just copy the GCBooterHomebrew folder into your /apps/ folder on your card and run it via the homebrew channel.

Requirements:
Wii (with the ability to boot some Wii homebrew)
Gamecube Controller

Source provided.
